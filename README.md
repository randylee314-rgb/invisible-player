## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://bedrock.makecode.com](https://bedrock.makecode.com)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/randylee314-rgb/invisible-player** and import

## Edit this project

To edit this repository in MakeCode.

* open [https://bedrock.makecode.com](https://bedrock.makecode.com)
* click on **Import** then click on **Import URL**
* paste **https://github.com/randylee314-rgb/invisible-player** and click import
